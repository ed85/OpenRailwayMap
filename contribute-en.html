OpenRailwayMap

Contribute

Daten erfassen und korrigieren

Das Erfassen von Daten is the most important way to contribute to this project. Das Projekt basiert auf dem Prinzip des "Crowdsourcing", das heißt, dass die Daten von unzähligen freiwilligen Mitarbeitern anhand von Ortskenntnis, Beobachtungen aus Spaziergängen und Zugfahrten oder mit der Hilfe von Luftbildern zusammengetragen, korrigiert und auf dem aktuellen Stand gehalten werden.

Um mitzumachen, ist nur ein normaler PC, ein kostenlos und frei erhältliches Bearbeitungsprogramm sowie ein wenig Grundwissen über die Eisenbahn notwendig. In einem tutorial werden die ersten Schritte ausführlich erklärt.

Donations

This project is completely run by volunteers without commercial aims. Providing this website generates running costs, which are funded partly by donations, partly by the maintainers of the project. We want to remain independent from commercial interests and provide this service without advertisement. Therefore we need your financial support.

Software development

Auch für Softwareentwickler gibt es bei der OpenRailwayMap jede Menge zu tun. Ob an der Datenbank, dem Kartenrendering, der Suchfunktion und diversen Hilfsprogrammen: You can resolve bugs, make improvements or implement new features. The code is managed in our Github repository.

Data imports

Hast Du Zugriff auf Daten, die wir verwenden dürfen? Solche Datenquellen können wir immer gut gebrauchen. Contact us and introduce yourself on our public mailing list. If you plan to import data yourself, you should consider the import guidelines of OpenStreetMap and have some mapping experience.

Translations

OpenRailwayMap is an international project. To make the project usable for users from different countries and help them to contribute to the project, it is aimed to translate the website, wiki pages and JOSM tagging presets to as many languages as possible. Insbesondere Leute mit Kenntnissen von bahnbezogenen Fachbegriffen in verschiedenen Sprachen sind hier gefragt.

Issues & Bug reports

Wenn du einen Fehler gefunden oder Ideen für Verbesserungen hast, dann nimm contact us or create an issue in our repository on Github.

Map Rendering Style, Tagging Schema & JOSM Tagging Presets

Arbeite mit an der Erweiterung und Verbesserung des Kartenstils. Or support us by creating country-specific tagging schemes and their corresponding tagging presets for JOSM editor.
